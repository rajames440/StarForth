\section*{Peer Review Submission Package Checklist (Superseded)}

This checklist assembled submission requirements for three target venues
(\textsc{ieee tse}, \textsc{toplas}, and \textsc{jss}), covering figure
formatting, supplementary material organization, and reviewer response
preparation. The submission it describes was subsequently completed and
the paper published on \textsc{ssrn}. This checklist is superseded by
that outcome. Future submissions to additional venues should prepare
fresh checklists against current venue guidelines.
