\section*{Physics Engine Validation Experiment Overview (November 2025)}

This document provided the quick-start reference for the comprehensive physics
engine validation experiment: a 90-run, three-configuration randomized trial
(A\_BASELINE, B\_CACHE, C\_FULL) designed to produce publication-grade
empirical evidence for the StarForth adaptive runtime. The experiment was
executed as described, and its results---including the finding of 0\%
algorithmic variance and \mbox{$-25.4\%$} convergence improvement under
the full-physics configuration---were reported in the published \textsc{ssrn}
paper. This overview is retained as a provenance record.
